Text formatting operations are done with a FF byte followed by an operation byte, plus any arguments.

FF xx ..


Valid operations:

  FF 80                 ; Newline.
  FF 81 ?? ??           ; Unknown. Dialogue pacing?
  FF 82 rr gg bb aa     ; Set text color. Each component is computed as value - 1. So for the brighest red, FF 82 00 01 01 00 becomes rgba(255,0,0,255).
  FF 83 ?? ?? ?? ?? ??  ; Unknown.
  FF 84 ?? ?? ??        ; Unknown.
  FF 85 ?? ??           ; Unknown.
  FF 87 ??              ; Unknown.
  FF 88 ss              ; Insert pixel spacing.
  FF 89 ??              ; Unknown.
  FF 8A ii              ; Glyph. 1-22 are valid.
  FF 8B ?? ??           ; Unknown.
  FF 8C ??              ; Unknown.
  FF 8D ?? ??           ; Unknown.

  FF 93                 ; Insert protagonist name.
  FF 94                 ; Insert order name.

  FF 9B                 ; Reset text color to the previous color.

  FF A9                 ; Unknown.
